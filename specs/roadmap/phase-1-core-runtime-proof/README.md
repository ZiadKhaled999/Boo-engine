# Phase 1: Core Runtime Proof

## Goal
Run a web app as a real Android app through Boo Engine.

## Required Outcomes
- Working APK/AAB package path.
- Working JS↔Native bridge call.
- Offline startup verified.
- Demo app that another developer can build.

## Delivery Strategy
- Build strictly as a vertical slice in stage order.
- Keep Boo core minimal and move feature logic into plugins.
- Validate one production-grade bridge path before expanding capability surface.
